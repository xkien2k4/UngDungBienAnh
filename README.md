Canny Edge Detection App (PyQt5 + OpenCV)

Ứng dụng phát hiện biên ảnh sử dụng thuật toán Canny Edge Detection được xây dựng bằng Python, kết hợp thư viện OpenCV và PyQt5.

👨‍💻 Author: Nguyễn Vũ Xuân Kiên

📌 Giới thiệu

Ứng dụng cho phép:

Chọn ảnh từ máy tính

Hiển thị ảnh gốc

Phát hiện biên bằng thuật toán Canny

Điều chỉnh 2 ngưỡng (threshold1, threshold2) bằng thanh trượt

Xem kết quả trực tiếp trên giao diện

Đây là một project nhỏ phục vụ học tập môn Xử lý ảnh.

pip install opencv-python PyQt5 numpy
